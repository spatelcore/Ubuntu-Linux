    $
    Try: ls -la then cd /var/log and cat syslog
    Pipes: cat notes.txt | sort | uniq -c
    grep: grep -i failed /var/log/syslog
    Redirect: echo hello > hi.txt then cat hi.txt
    find: find / -name "*.txt"
    Reset: type reset to restore the filesystem

    Exam outline

    The Using Linux Terminal exam validates your ability to secure, operate, and manage basic system resources. Reference: canonical.com/academy/exam-content

    Section 1 — Navigating Files and Filesystems

    • Navigate and manipulate directories and files with terminal commands
    • Search, compare, and modify files using regular expressions, pipe operators, and redirection

    Section 2 — Managing System Resources

    • Locate system logs and configure log rotation
    • Reconstruct disk partitions and filesystems using fdisk, fsck, parted
    • Identify correct crontab format and location
    • Interpret system logs during troubleshooting

    Section 3 — Securing Filesystem Access

    • Create and manage SSH keys
    • Configure system-wide and user-specific security settings
    • Configure password complexity and expiry rules
    • Interpret sudo configuration policies and their security implications
    • Manage user and group accounts, access, and membership
    • Manage directory and file ownership and access
    Note: "Interpret common open-source licenses" is currently in this exam but moves to the Using Desktop exam in the 2026 edition.
